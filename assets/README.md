## Naming Convention

- lowercase
- kebab-case
- purpose-first names
- examples:
  - `shiran-teaching-workshop.jpg`
  - `parent-practicing-wrap.jpg`
  - `wrap-fabric-closeup.jpg`
  - `workshop-atmosphere.jpg`

## Suggested Dimensions

- Hero or large editorial image:
  - `1600px` to `2200px` wide
- Section image cards:
  - `1200px` to `1600px` wide
- Detail crops / closeups:
  - at least `1000px` on the long edge
- Use modern aspect ratios when possible:
  - `4:5`
  - `3:4`
  - `16:10`

## Compression Notes

- Prefer compressed JPEG or WebP for photos.
- Keep files visually clean but lightweight enough for static GitHub Pages delivery.
- Target roughly:
  - hero images under `350 KB` to `500 KB` when possible
  - section images under `250 KB` to `350 KB` when possible
- Export at the final display size instead of uploading oversized originals.

## Usage Notes

- Use real images only.
- Prefer candid, calm, documentary-style workshop imagery over posed stock-looking images.
- If a future video poster is added, place it under `assets/images/` and name it by section/purpose.
- If future short local clips are added, place them under `assets/videos/`.

## Important Warning

- Do not add fake testimonials.
- Do not add fake review screenshots.
- Do not add AI-generated human workshop photos.
- AI-generated abstract backgrounds are acceptable only if clearly decorative and not presented as documentary proof.
